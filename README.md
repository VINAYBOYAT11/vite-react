## 🔍 About

This project is a **modern React application** built using [Vite](https://vitejs.dev) — the next-generation frontend build tool that offers:

- ⚡ **Instant server start** — No waiting for webpack bundle
- 🔥 **Hot Module Replacement (HMR)** — Instant UI updates on save
- 📦 **Optimized build** — Rollup-based production bundling
- 🔷 **TypeScript support** — Full type safety

This serves as a template and learning ground for building modern React applications with best practices.

## 🚀 Getting Started

### Prerequisites

```bash
node --version    # v18.0.0 or higher
npm --version     # v9.0.0 or higher
```

### Installation

```bash
# 1. Clone the repository
git clone https://github.com/VINAYBOYAT11/vite-react.git
cd vite-react

# 2. Install dependencies
npm install

# 3. Start the development server
npm run dev
```

Visit `http://localhost:5173` in your browser 🎉

## 🚢 Deployment

This app is deployed on **Vercel**. To deploy your own:

```bash

npm i -g vercel

# Deploy
vercel

# Or connect your GitHub repo to Vercel dashboard for auto-deploy on push
```
